torchnlp.text\_encoders package
===============================

The ``torchnlp.text_encoders`` package supports encoding text :class:`str` as a vector
:class:`torch.Tensor` and decoding a vector :class:`torch.Tensor` back to :class:`str`. This package
was inspired by Googles Tensor2Tensor repository TextEncoder class.
